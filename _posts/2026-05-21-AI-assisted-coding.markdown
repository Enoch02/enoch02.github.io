Once in a while, I use AI to figure out a possible solution to whatever coding problem I'm stuck on. I usually achieve this by selecting the relevant code block and pasting it into the web interface of my LLM of choice.

Recently, I decided to install Gemini's CLI via Homebrew on my computer. The experience has been quite eye-opening, and I keep wondering why I had not done this earlier.

I used it to analyse the HTML code of a site I wanted to scrape some links from using [skrape-it](https://github.com/skrapeit/skrape.it). It saved me a lot of time; I would have had to go through thousands of lines of HTML to understand the structure and then create a query that extracts the data I want.

I don't want to rely too heavily on AI-generated code, so I take my time to read the documentation whenever the AI calls some method I have not seen before or uses some unorthodox approach to solving a certain problem.
